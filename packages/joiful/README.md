# TypePoint Joiful

Joiful-based validation for TypePoint.

## Install

Install `@typepoint/joiful` and it's peer dependencies.

```sh
npm add @typepoint/joiful joiful  @typepoint/server @typepoint/shared reflect-metadata
```
